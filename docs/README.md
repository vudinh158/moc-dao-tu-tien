# 📂 Docs - Mộc Đạo Tu Tiên

Thư mục này chứa các tài liệu liên quan đến quá trình phân tích, thiết kế và báo cáo của dự án **Mộc Đạo Tu Tiên**.

## Các tài liệu hiện có
- **BRD.md (Business Requirements Document):** Tài liệu yêu cầu nghiệp vụ, định nghĩa lại mục tiêu hệ thống gamification kết hợp IoT.
- **PRD.md (Product Requirements Document):** Tài liệu yêu cầu sản phẩm, mô tả các luồng nghiệp vụ và tính năng.
- **System_Design.md:** Tài liệu thiết kế kỹ thuật, bao gồm Database Schema, API và các thuật toán cốt lõi.
- **Hardware_Spec.md:** Đặc tả phần cứng và các cảm biến.
- **Slides báo cáo:** Các file `.pptx` dành cho báo cáo chuyên môn và kĩ thuật.
