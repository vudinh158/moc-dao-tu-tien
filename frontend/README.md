# 📂 Frontend - Mộc Đạo Tu Tiên

Thư mục này chứa mã nguồn Frontend cho Web App của dự án **Mộc Đạo Tu Tiên**.

## Chức năng chính
- Giao diện người dùng chăm sóc cây.
- Theo dõi trạng thái cây, chỉ số môi trường (độ ẩm, ánh sáng).
- Hệ thống Cảnh Giới và Bảng Xếp Hạng.
- Giao diện dành cho Admin (Dashboard quản lý hệ số Tu Vi, quản lý thiết bị).

*(Chi tiết công nghệ và hướng dẫn cài đặt khởi chạy project sẽ được cập nhật sau)*
